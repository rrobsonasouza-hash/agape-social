\# Design System — Ágape



Versão: 1.0



\---



\# Objetivo



O Design System do Ágape define os padrões visuais e de experiência do usuário para toda a plataforma.



Seu objetivo é garantir:



\- Consistência visual

\- Facilidade de manutenção

\- Experiência intuitiva

\- Reutilização de componentes

\- Rapidez no desenvolvimento



Toda nova tela deverá seguir este documento.



\---



\# Princípios



\## Simplicidade



O sistema deve ser fácil de utilizar mesmo por usuários com pouca familiaridade com tecnologia.



Evitar excesso de informações.



\---



\## Clareza



A informação mais importante sempre deve aparecer primeiro.



\---



\## Acolhimento



A interface deve transmitir serenidade, organização e confiança.



Evitar excesso de cores fortes.



\---



\## Consistência



Um componente deve possuir sempre o mesmo comportamento em toda a plataforma.
